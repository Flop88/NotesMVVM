**Приложение Заметки**

> В приложении вы выбираете где будут храниться все ваши данные - на сервере или в локальной БД

**Архитектурные компоненты:**
- LiveData;
- ViewModel;
- Room;
- Navigation;

**Стек технологий:**
- Android SDK;
- Kotlin;
- Шаблон MVVM;
- Kotlin Coroutines;
- Firebase SDK;
- RecyclerView.
